# 설정 변경 가이드 (값 바꾸는 법)

값은 모두 `AAA` 자리표시자로 넣어두었습니다. **메모장으로 파일을 열고 `value="AAA"` 의 `AAA` 만 실제 값으로 바꾸면** 됩니다.

---

## 1. AVEVA 설치 경로 (이미 설정됨 — 보통 안 건드림)

- 파일: **`Directory.Build.props`** (압축 푼 폴더 맨 위)
- 현재 값: `C:\AVEVA\Marine\OH12.1.SP5\`
- 다를 때만 그 한 줄(`AvevaBinDir`)의 경로를 수정 (끝에 `\` 유지)

---

## 2. 로그인/모듈 값 — 어떤 파일을 고치나? (실행 방식에 따라 다름)

### A) 웹판으로 실행할 때 (`run-web.cmd` 더블클릭)
- **로그인(PROJECT / USER / PASSWORD / MDB)은 파일이 아니라 브라우저 화면에서 입력합니다.** 파일 수정 불필요.
- 단, **모듈 번호와 환경변수**만 이 파일에서:
  - 파일: **`E3dLeafWeb\E3dLeafWeb.Standalone\App.config`**
  - `MODULE_NUMBER` 의 `value="AAA"` → AM Outfitting 모듈 번호
  - `projects_dir`, `AVEVA_DESIGN_USER`, `PMLLIB`, `AAAA000` → 실제 Marine 환경 경로 (`marine-probe.ps1` 참고)

### B) 콘솔판으로 실행할 때 (`run-console.cmd` 더블클릭)
- 파일: **`E3dLeafExport\E3dLeafExport\App.config`**
- 이 부분의 `value="AAA"` 를 실제 값으로:
  ```
  <add key="PROJECT"  value="AAA" />   ← 프로젝트 코드
  <add key="USER"     value="AAA" />   ← 사용자
  <add key="PASSWORD" value="AAA" />   ← 비밀번호 (없으면 value="")
  <add key="MDB"      value="AAA" />   ← MDB 이름
  <add key="MODULE_NUMBER" value="AAA" />  ← AM Outfitting 모듈 번호
  ```
- 환경변수(`projects_dir` 등)도 실제 경로로.

---

## 3. 바꾸는 방법 (메모장)

1. 위 파일을 **우클릭 → 연결 프로그램 → 메모장**으로 열기
2. `value="AAA"` 에서 **따옴표 `" "` 사이의 `AAA` 만** 실제 값으로 교체
   - 예: `value="AAA"` → `value="MyProj"`
   - 비밀번호 없으면: `value=""` (빈칸)
3. **저장** (Ctrl+S)
4. `run-web.cmd` 또는 `run-console.cmd` 더블클릭

> ⚠️ `key="..."` 부분(왼쪽)은 건드리지 마세요. **오른쪽 `value="..."` 안의 AAA만** 바꿉니다.

---

`78` 은 E3D Model 용이라 Marine 은 다릅니다. AM Outfitting 모듈 번호는:

- 또는 AVEVA 관리자에게 문의

값을 모르면 일단 비워두지 말고 `78` 로 시도해보고, 안 되면 marine-probe 결과를 공유해 주세요.
